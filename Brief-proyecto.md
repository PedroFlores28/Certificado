# Ficha del Proyecto: Sistema de Badges Digitales para Embajadores UPC v2.0
Versión: Evaluación SaaS vs. Implementación Open-Source
Cliente: UPC
Objetivo: Implementar un sistema para emitir badges digitales verificables para el programa Embajadores UPC.
Version: 2.0

## 1. Descripción del Proyecto
La institución busca implementar un sistema de credenciales digitales (badges) que permita reconocer a los Embajadores UPC con un badge que es un diploma, compatible con LinkedIn y con metadatos .

El sistema debe permitir emitir, gestionar, verificar y compartir badges de forma institucional de forma visual

## 2. Requerimientos del Cliente
Requerimientos Funcionales
Emisión de badges digitales con metadatos completos.
Plantillas con branding institucional UPC.
URL verificable públicamente.
Compatibilidad con LinkedIn (sección Licenses & Certifications).
Panel para:
Emisión individual o masiva
Revocación
Registro histórico
Requerimientos No Funcionales
Seguridad y protección de datos.
Alta disponibilidad.
Experiencia de uso simple para administradores.
Bajos costos operativos.


## Resumen
Debe crear un admin y un perfil publico del diploma (badge) cuyo contenido tenemos, para que lo pueda subir a linked in.
En esta primera etapa no tiene que cumplir con el protocolo de e-certs, pero simularlo, solo un lugar donde se pueda ver el diploma digital, tener un id único, y las historias de usuario para los admin.
